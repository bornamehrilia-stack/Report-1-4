### ۱. هدف آزمایش

بررسی نحوه عملکرد ورودی‌های دیجیتال و استفاده از مقاومت‌های خارجی برای تعیین وضعیت منطقی دقیق پین در هنگام باز بودن کلید .

### ۲. شرح مختصر مدار

در این مدار، یک LED به پین شماره **۲** متصل شده است. همچنین یک کلید به پین شماره **۸** متصل است که با یک مقاومت به زمین (GND) وصل شده (آرایش Pull-down). زمانی که کلید فشرده شود، ولتاژ ۵ ولت به پین ۸ می‌رسد و برنامه دستور روشن شدن LED را صادر می‌کند. در صورت رها بودن کلید، مقاومت باعث می‌شود پین ورودی در وضعیت صفر منطقی باقی بماند.

### ۳. موارد مورد نیاز

* ۱ عدد برد **Arduino Uno**
* ۱ عدد **LED**
* ۱ عدد کلید فشاری (**Push Button**)
* ۲ عدد **مقاومت** (یکی برای LED و دیگری برای دکمه)
* برد بورد و سیم‌های اتصال

### ۴. نمایش کد پروژه

```cpp
int buttonPin = 8;    // تعریف پین ورودی دکمه
int ledPin = 2;       // تعریف پین خروجی ال‌دی‌دی
int buttonState = 0;  // متغیر ذخیره وضعیت دکمه

void setup() {
  pinMode(ledPin, OUTPUT); // تنظیم پین 2 به عنوان خروجی
  pinMode(buttonPin, INPUT); // تنظیم پین 8 به عنوان ورودی
}

void loop() {
  // خواندن وضعیت فعلی دکمه
  buttonState = digitalRead(buttonPin);

  // بررسی شرط: اگر دکمه فشرده شود (HIGH)
  if(buttonState == LOW){
    digitalWrite(ledPin, HIGH); // روشن کردن LED
  }
  else{
    digitalWrite(ledPin, LOW);  // خاموش کردن LED
  }
}

```

### ۵. توضیح مختصر کد

* **تعریف متغیرها:** پین‌های مورد نظر نام‌گذاری شده‌اند تا خوانایی کد بالا برود.
* **تابع `digitalRead`:** این تابع وضعیت پین ۸ را بررسی می‌کند. با توجه به کد شما، منطق بر اساس `LOW` بودن تنظیم شده است (اگر دکمه در حالت عادی `HIGH` باشد، با فشردن و رسیدن به `LOW` چراغ روشن می‌شود).
